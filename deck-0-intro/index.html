AOP vs SOP

SOP — Para humanos
  • Standard Operating Procedure
  • Paso a paso para el equipo
  • Capacitacion, seguimiento, actualizaciones
  • Adopcion de abajo hacia arriba: la gente necesita soporte
AOP — Para agentes AI
  • Agentic Operating Procedure
  • Rutina + reglas de escalacion para el AI
  • MCP tools como el vocabulario del agente
  • Humano en el ciclo: el AI escala, el humano decide
"No tiene sentido ensenar a escribir icebreakers a escala — eso ya es un AOP. El SOP es para lo que aun requiere juicio humano."

CRM Legacy
  • APIs complejas, limites de uso agresivos
  • Requiere admin dedicado
  • No soporta MCP nativamente
  • Excesivo para equipos <50 personas
  • El AI no puede operar libremente

  • API-first, listo para MCP
  • Se configura en horas, no meses
  • Sincronizacion nativa con email, calendario, LinkedIn
  • El AI lee y escribe como un usuario mas
  • Pensado para equipos agiles

Las 4 Dimensiones del Scoring

Mas dinero con menos esfuerzo: pon tus mejores recursos en las cuentas de mayor valor

1
Fit de Cuenta
ICP scoring: firmografico, tecnografico, headcount, funding
2
Fit Individual
Seniority, departamento, autoridad de compra — A/B/C/D
3
Senales de Intencion
Visitas al sitio, registros a webinars, descargas de contenido, interacciones sociales
4
Senales Declaradas
Presupuesto, timeline, criterios de decision — lo que revelan en reuniones

Cada dimension tiene un peso diferente segun tu negocio. Los pesos se calibran con el equipo.

El Camino ABM: de lo amplio a lo especifico

No empiezas con mensajes personalizados por cuenta. Empiezas con grupos.

1
Account Tiering (A/B/C/D)
3-4 grupos. Tier 1 → AE + cobertura publicitaria generosa. Tier 2 → BDR + incentivo. Tier 3 → PLG.
2
Individual Tiering
Dentro de cada cuenta: es la persona correcta? Seniority, departamento, autoridad de compra.
3
Behavioral Scoring
Senales de intencion: que estan haciendo? Descargaron contenido? Visitaron la pagina de precios?
4
Senales Declaradas + Mensajes Personalizados
Solo aqui inviertes en mensajes personalizados por cuenta. Necesitas el scoring robusto primero.
