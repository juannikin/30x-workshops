Delivery & Booking

Email via Instantly (warmup, domain rotation). LinkedIn via Unipile API. Calendar booking con intent detection.

Sourcing & Scoring tu TAM

Dimensiones del Scoring

  • Firmographic: headcount, revenue, industry
  • Technographic: usan competidores? Que stack?
  • Demographic: geo, idioma, timezone
  • Psychographic: hiring signals, growth rate

Cada dimension tiene un peso (1-10). Los pesos se calibran con tu equipo — no se imponen.

Fuentes

  • Tu CRM existente — scoring de lo que ya tienes
  • Clay — waterfall enrichment, contact info
  • Exa — "encuentra empresas que se parecen a mis mejores clientes"

Tiering A/B/C/D + Routing

Tier 1 — Gold

  • Directo al AE
  • Ad cover generoso (LinkedIn, Google)
  • Outreach personalizado multi-canal
  • "Tomalos al palco VIP"

Tier 2 — Silver

  • Ruta a BDR
  • CTA: "Talk to a human"
  • Incentivo: headphones, free audit
  • Sequence standard + LinkedIn

Tier 3 — PLG

  • Self-serve path
  • Nurture sequences
  • Content + webinars
  • Score up → upgrade tier

El LTV dicta el CAC que puedes pagar. Tier 1 merece mas inversion por deal.

Copy Generation con Claude

Personalizacion a escala — diferente por tier, persona y signal

Principios

  • Tone calibrado por tier (formal vs casual)
  • Variables: {{firstname}}, {{company_pain}}
  • Cada email tiene un hook basado en data real
  • No icebreakers genericos — son AOPs ahora

Variables disponibles

{{firstname}} {{company_name}} {{company_pain}} {{mutual_connection}} {{recent_news}} {{tech_stack}} {{hiring_signal}} {{tier}}

Anatomia de un email que convierte

Subject: {{recent_news}} + quick thought

Hi {{firstname}},

Saw that {{company_name}} just {{recent_news}}. ← Hook basado en data real

We've been helping companies like {{similar_customer}} solve exactly this — {{company_pain}}. ← Value prop + social proof

In their case, {{result_metric}}. ← Resultado concreto

Worth a 15-min call this week? ← CTA claro y corto

— Juan

5 lineas. 1 hook. 1 value prop. 1 CTA. No mas.

Por que omnichannel?

  • Email solo: 15-25% open rate
  • Email + LinkedIn: 40-60% touch rate
  • Cada canal refuerza el otro
  • LinkedIn humaniza, email escala

Contingencias

  • Si acepta LinkedIn → skip email 2
  • Si responde email → pausa sequence
  • Si positive reply → trigger booking flow
  • Si negative → graceful exit

Instantly (Email) + Unipile (LinkedIn)

Instantly

  • Warmup automatico de dominios
  • Rotacion de dominios y mailboxes
  • Deliverability monitoring
  • A/B testing de subject lines
  • API para integracion con Claude Code

Mejor que Outreach para startups: setup en minutos, pricing accesible.

Unipile API (LinkedIn)

  • Connection requests programaticos
  • Messaging nativo dentro de LinkedIn
  • Mismo approach que OpenClaw
  • Templates con variables
  • Compliance: respetar limits de LinkedIn

No es scraping — es API authorizada. El mensaje llega donde el prospect ya esta.

Intent Classification

  • Positive: "interested", "let's talk", "tell me more"
  • Question: "how does it work?", "pricing?"
  • Negative: "not interested", "remove me"
  • OOO: auto-reply detected → reschedule

El AOP del BDR Agent

Rutina Diaria

  • 7:00 AM — Revisar replies nuevos → classify_reply() → responder o escalar
  • 8:00 AM — Nuevos leads en pipeline → enrich_lead() → score_lead()
  • 9:00 AM — Generar copy para leads Tier 1-2 → generate_sequence()
  • 10:00 AM — Enviar emails + LinkedIn touches del dia
  • 2:00 PM — Segundo check de replies → booking para positivos
  • 5:00 PM — Reporte en Slack: actividad del dia, meetings booked, escalaciones

Escalar al humano cuando:

  • Lead es Tier 1 (high-value account)
  • Sentiment negativo o confuso
  • Pregunta compleja de pricing/scope

El AI maneja solo:

  • Enrichment + scoring de nuevos leads
  • Copy generation para Tier 2-3
  • Booking de meetings confirmados
