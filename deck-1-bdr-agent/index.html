Sourcing & Scoring tu TAM

Dimensiones del Scoring

  • Firmografico: headcount, revenue, industria
  • Tecnografico: usan competidores? Que stack?
  • Demografico: geo, idioma, zona horaria
  • Psicografico: senales de contratacion, tasa de crecimiento

Cada dimension tiene un peso (1-10). Los pesos se calibran con tu equipo — no se imponen.

Fuentes

  • Tu CRM existente — scoring de lo que ya tienes
  • Clay — enriquecimiento en cascada, info de contacto
  • Exa — "encuentra empresas que se parecen a mis mejores clientes"

Tiering A/B/C/D + Routing

Tier 1 — Gold

  • Directo al AE
  • Cobertura publicitaria generosa (LinkedIn, Google)
  • Outreach personalizado multicanal
  • "Tomalos al palco VIP"

Tier 2 — Silver

  • Ruta a BDR
  • CTA: "Habla con un humano"
  • Incentivo: audifonos, auditoria gratis
  • Secuencia estandar + LinkedIn

Tier 3 — PLG

  • Ruta de autoservicio
  • Secuencias de nurture
  • Contenido + webinars
  • Sube el score → sube de tier

El LTV dicta el CAC que puedes pagar. Tier 1 merece mas inversion por deal.

Generacion de Copy con Claude

Personalizacion a escala — diferente por tier, persona y senal

Principios

  • Tono calibrado por tier (formal vs casual)
  • Variables: {{firstname}}, {{company_pain}}
  • Cada email tiene un hook basado en data real
  • No icebreakers genericos — son AOPs ahora

Variables disponibles

{{firstname}} {{company_name}} {{company_pain}} {{mutual_connection}} {{recent_news}} {{tech_stack}} {{hiring_signal}} {{tier}}

Anatomia de un email que convierte

Por que omnichannel?

  • Email solo: 15-25% open rate
  • Email + LinkedIn: 40-60% touch rate
  • Cada canal refuerza el otro
  • LinkedIn humaniza, email escala

Contingencias

  • Si acepta LinkedIn → salta email 2
  • Si responde email → pausa secuencia
  • Si respuesta positiva → activa flujo de agendamiento
  • Si negativa → salida elegante

Instantly (Email) + Unipile (LinkedIn)

Instantly

  • Warmup automatico de dominios
  • Rotacion de dominios y mailboxes
  • Monitoreo de entregabilidad
  • A/B testing de lineas de asunto
  • API para integracion con Claude Code

Mejor que Outreach para startups: configuracion en minutos, precio accesible.

Unipile API (LinkedIn)

  • Solicitudes de conexion programaticas
  • Mensajeria nativa dentro de LinkedIn
  • Mismo enfoque que OpenClaw
  • Plantillas con variables
  • Cumplimiento: respetar limites de LinkedIn

No es scraping — es API autorizada. El mensaje llega donde el prospecto ya esta.

Clasificacion de Intencion

  • Positiva: "me interesa", "hablemos", "cuentame mas"
  • Pregunta: "como funciona?", "precios?"
  • Negativa: "no me interesa", "retirame"
  • Fuera de oficina: auto-respuesta detectada → reprogramar

Flujo de Agendamiento

  • Google Calendar API o Calendly
  • Sugerir automaticamente 3 horarios disponibles
  • Confirmacion + invitacion de calendario automatica
  • Recordatorio 24h y 1h antes

El AOP del BDR Agent

Rutina Diaria

  • 7:00 AM — Revisar respuestas nuevas → classify_reply() → responder o escalar
  • 8:00 AM — Nuevos leads en pipeline → enrich_lead() → score_lead()
  • 9:00 AM — Generar copy para leads Tier 1-2 → generate_sequence()
  • 10:00 AM — Enviar emails + contactos LinkedIn del dia
  • 2:00 PM — Segunda revision de respuestas → agendamiento para positivos
  • 5:00 PM — Reporte en Slack: actividad del dia, reuniones agendadas, escalaciones

Escalar al humano cuando:

  • Lead es Tier 1 (cuenta de alto valor)
  • Sentimiento negativo o confuso
  • Pregunta compleja de precios/alcance

El AI maneja solo:

  • Enrichment + scoring de nuevos leads
  • Generacion de copy para Tier 2-3
  • Agendamiento de reuniones confirmadas
